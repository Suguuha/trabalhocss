Universidade Paulista de Tecnologia (UPT)

A UPT é uma das maiores universidades da américa do sul. Além de oferecer cursos de graduação presencial e EAD, também oferecemos cursos de pós graduação, mestrado, doutorado e cursos de extensão.

Cursos de extensão oferecidos pela UPT:

      Gestão de TI
      SQL Fundamentos.
      Hardware
      Programação em Python
      HTML e CSS

Gestão de TI

O foco principal da gestão de TI, é possibilitar o profissional de melhorar e otimizar seus processos e procedimentos que envolvem a área de TI.

Lógica de programação

Ajudar, ensinar ou aperfeiçoar seus conhecimentos com lógica de programação. A lógica de programação é o modo como se escreve um programa de computador, um algoritmo.

Hardware

Você aprenderá como realizar a manutenção do seu computador, notebook, impressora, celulares. Além disso, também aprenderá robótica, montagem de computadores e muito mais!

Programação em Python

O básico e intermediário da programação em Python, aprender uma linguagem de programação é sempre um aperfeiçoamento para o seu curriculo.

HTML e CSS

Aprenda como desenvolver os seus próprios sites!
